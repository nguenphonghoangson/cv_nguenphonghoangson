# 🎯 CÁC MẪU CV ĐÃ TẠO

## 📁 Danh Sách Files

### 1. **index.html** - CV Chính (Recommended)
- ✨ Thiết kế hiện đại, đầy đủ tính năng
- 🎨 Có dark mode, edit mode
- 📱 Responsive hoàn hảo

## 🚀 BẮT ĐẦU NHANH

### Bước 1: Chọn Template
- **Dùng CV đầy đủ tính năng**: Mở `index.html`
- **Dùng CV tối giản**: Mở `minimalist.html`

### Bước 2: Tùy Chỉnh
- Đọc file `QUICK_GUIDE.md` để hướng dẫn chi tiết
- Hoặc click nút "Chỉnh sửa" trên web

### Bước 3: Lưu & Chia Sẻ
- In thành PDF
- Hoặc deploy lên web

## ⚡ THAY ĐỔI NHANH NHẤT

### Cách 1: Chỉnh sửa trực tiếp trên web
1. Mở `index.html` bằng trình duyệt
2. Click nút "Chỉnh sửa" (góc trên phải)
3. Click vào text để sửa
4. Click "Lưu"

### Cách 2: Sửa file HTML
1. Mở `index.html` bằng Notepad/VSCode
2. Tìm và thay thế text
3. Lưu file
4. Refresh trình duyệt

## 🎨 CUSTOM MÀU SẮC NHANH

Mở `style.css`, tìm dòng `:root` và chọn theme:

### Blue Professional (Mặc định)
```css
--primary-color: #2c3e50;
--secondary-color: #3498db;
--accent-color: #e74c3c;
```

### Green Fresh
```css

--secondary-color: #2ecc71;
--accent-color: #f39c12;
```

### Purple Creative
```css
--primary-color: #8e44ad;

--accent-color: #e74c3c;
```

### Black Minimal
